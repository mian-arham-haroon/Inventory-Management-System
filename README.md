# 🛒 Inventory Management System

A professional, user-friendly inventory management application built with **Python Tkinter**. Effortlessly manage employees, products, suppliers, categories, and taxes through an intuitive GUI—ideal for small businesses.

---

## 🚩 Features

- 🔐 **Admin Login**
- 👥 **Employee Management**
- 📦 **Product Management**
- 🏢 **Supplier Management**
- 🗂️ **Category Management**
- 💸 **Tax Configuration**
- 📊 **Dashboard Overview**
- 🗄️ **SQLite Database**
- 🖼️ **Image Uploads**
- 📤 **Excel Export** *(optional)*

---

## 🛠️ Tech Stack

- 🐍 Python 3.8+
- 🖼️ Tkinter
- 🗄️ SQLite
- 🖼️ Pillow
- 📊 openpyxl *(optional)*

## 🚀 Quick Start

1. **Install dependencies:**
    ```bash
    pip install pillow openpyxl
    ```
2. **Run the application:**
    ```bash
    python main.py
    ```
